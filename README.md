## About

SCH Tools exists because the security awareness and human risk space is full of vendor-produced resources with strings attached. These tools have none. They run entirely in the browser, store nothing, and are free to use and share.

Each tool is a single self-contained HTML file. No frameworks, no build pipeline, no dependencies to manage beyond what's loaded from CDN where noted.
